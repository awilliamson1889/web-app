# Web-app
# Run

```{bash}
$ cd flask_app
$ export FLASK_APP=department_app
$ export FLASK_ENV=development
$ flask run
```
Or on Windows cmd:
```{bash}
> cd flask_app
> set FLASK_APP=department_app
> set FLASK_ENV=development
> flask run
```
Open http://127.0.0.1:5000 in a browser.
